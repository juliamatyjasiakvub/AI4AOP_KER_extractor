# AI4AOP_KER_extractor
Generate a ready to use rubric for KEs and related KER based on open access and user's papers

## Link
https://ai4aop-ker-extractor.streamlit.app/

## How to use
Currently, 3 modules are available:

### Stage 1 — Search & screen
Supports your PubMed search by pre-screening abstracts according to your set criteria

### Stage 2 — KER extraction
Upload your papers to extract studied biological changes to enrich your AOP with KEs/KERs

### Pathway Visualization
Visualise the data from Stage 2
